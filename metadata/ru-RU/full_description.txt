GitTasks — это приложение для Android для ведения заметок и управления задачами с интеграцией Git, оптимизированное для эффективного управления заметками на основе Markdown с расширенными функциями, такими как фильтрация тегов, просмотр журнала Git и бесшовная синхронизация.

Функции включают:

- создание/открытие/клонирование репозиториев
- поддержка приватных репозиториев (SSH и HTTPS)
- заметки и элементы задач
- поиск (глобальный и по конкретным папкам)
- отображение в виде сетки с настраиваемыми ширинами
- отображение в виде списка
- режим редактирования Markdown и полное рендеринг Markdown
- кликабельные чекбоксы в отображениях и рендеринг Markdown
- удалённая синхронизация
- сортировка по времени
- поддержка нескольких языков
- гибкое отображение тегов в сетке и списке
- фильтрация и организация на основе тегов
- поддержка метаданных frontmatter
- просмотрщик журнала Git
- автоматическое разрешение конфликтов слияния
- управление активами для изображений и файлов
