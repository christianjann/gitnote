GitTasks is an Android note-taking and task management app with Git integration optimized for efficient markdown-based note management with advanced features like tag filtering, Git log viewing, and seamless synchronization.

Features include:

- create/open/clone repositories
- private repo support (SSH and HTTPS)
- note and task items
- search (global and from specific folder)
- grid view with customizable widths
- list view
- markdown edit view and full markdown rendering
- clickable check boxes in views and markdown rendering
- remote sync
- time based sort
- multi-language support
- flexible tag display in grid and list view
- tag-based filtering and organization
- frontmatter metadata support
- git log viewer
- automatic merge conflict resolution
- asset management for images and files
