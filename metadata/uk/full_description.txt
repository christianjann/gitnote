GitTasks — це додаток для Android для ведення нотаток та управління завданнями з інтеграцією Git, оптимізований для ефективного управління нотатками на основі Markdown із розширеними функціями, такими як фільтрація тегів, перегляд журналу Git та безшовна синхронізація.

Функції включають:

- створення/відкриття/клонування репозиторіїв
- підтримка приватних репозиторіїв (SSH та HTTPS)
- нотатки та елементи завдань
- пошук (глобальний та по конкретних папках)
- відображення у вигляді сітки з налаштовуваними ширинами
- відображення у вигляді списку
- режим редагування Markdown та повне рендеринг Markdown
- клікабельні чекбокси у відображеннях та рендеринг Markdown
- віддалена синхронізація
- сортування за часом
- підтримка кількох мов
- гнучке відображення тегів у сітці та списку
- фільтрація та організація на основі тегів
- підтримка метаданих frontmatter
- переглядач журналу Git
- автоматичне вирішення конфліктів злиття
- управління активами для зображень та файлів
